EllipticalOrbitAnimation3D(focalPoint: Vector3!, periapsis: Vector3!, normal: Vector3!, eccentricity: Double, direction: EllipticalOrbitAnimation3D.OrbitDirection!)

Defines an elliptical orbit around a point.

Parameters

focalPoint - Vector3!: Point which the ATransformable3D orbits around.

periapsis - Vector3!: Point which the object passes closest to the focal point.

normal - Vector3!: Normal to the orbital plane. This defines the orbital inclination.

eccentricity - Double: Eccentricity of the orbit. Zero value results in a circular orbit.

direction - EllipticalOrbitAnimation3D.OrbitDirection!: Direction of the orbit.

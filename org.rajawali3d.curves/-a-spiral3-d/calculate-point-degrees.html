calculatePointDegrees

open fun calculatePointDegrees(result: Vector3!, theta: Double): Unit

Calculates the position on the spiral for the specified polar angle. This takes an additional parameter of a Vector3 which will be set to the calculated position.

Parameters

result - Vector3!: Vector3 to set with the updated position.

theta - Double: double the polar angle to calculate for, in degrees.

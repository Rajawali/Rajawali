ArchimedeanSpiral3D(scale: Double, density: Double, start: Vector3!, normal: Vector3!, spiralIn: Boolean)

Constructs a ArchimedeanSpiral3D with the specified parameters.

Parameters

scale - Double: double Factor which determines the overal size of the spiral.

density - Double: double Factor which determines how tightly the spiral is curled. Smaller values result in tighter curling.

start - Vector3!: Vector3 The point where the spiral should start from.

normal - Vector3!: Vector3 The normal vector of the plane the spiral is in. This is assumed to be orthogonal to the vector formed from the start to the origin.

spiralIn - Boolean: boolean True if the spiral should move from the staring point in. False to move from starting point out. Note that for an Archimedean spiral, inward spiraling is not infinite as is the case for a logarithmic spiral. A sufficient number of rotations will cause a "loop" to form and it will begin spiraling outward.

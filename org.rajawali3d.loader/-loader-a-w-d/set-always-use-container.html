setAlwaysUseContainer

open fun setAlwaysUseContainer(flag: Boolean): Unit

Determine if #getParsedObject() will force the model to use a container. When more than one model is in an AWD file, the models will be wrapped in a container, this flag can force single containers to be wrapped in a container as well.

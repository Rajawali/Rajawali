class Lambert : IDiffuseMethod
class Lambert : IDiffuseMethod

Defines Lambertian reflectance. This technique causes polygons to reflect light equally in all directions. The reflection is calculated by taking the dot product of the surface's normal vector and a normalized light direction vector. This number is then multiplied by the color of the surface and the intensity of the light hitting the surface.

To use the Lambertian diffuse method you need to create an instance of one of the methods and then assign it to a material using the Material#setDiffuseMethod(IDiffuseMethod) method: 


      material.setDiffuseMethod(new DiffuseMethod.Lambert());

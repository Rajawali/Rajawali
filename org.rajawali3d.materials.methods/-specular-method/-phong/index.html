Phong

class Phong : ISpecularMethod
class Phong : ISpecularMethod

Phong shading interpolates surface normals across rasterized polygons and computes pixel colors based on the interpolated normals and a reflection model. To use the Phong specular method you need to create an instance of one of the methods and then assign it to a material using the Material#setSpecularMethod(ISpecularMethod) method: 


	  material.setSpecularMethod(new SpecularMethod.Phong());

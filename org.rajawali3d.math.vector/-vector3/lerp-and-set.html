@NonNull open fun lerpAndSet(@NonNull from: Vector3, @NonNull to: Vector3, amount: Double): Vector3

Performs a linear interpolation between from and to by the specified amount. The result will be stored in the current object which means that the current x, y, z values will be overridden.

Parameters

from - Vector3: Vector3 Starting point.

to - Vector3: Vector3 Ending point.

amount - Double: double [0-1] interpolation value.

Return
Vector3: A reference to this Vector3 to facilitate chaining.

Plane

open class Plane : Object3D
open class Plane : Object3D

A plane primitive. The constructor takes two boolean arguments that indicate whether certain buffers should be created or not. Not creating these buffers can reduce memory footprint.

When creating solid color plane both createTextureCoordinates and createVertexColorBuffer can be set to false.

When creating a textured plane createTextureCoordinates should be set to true and createVertexColorBuffer should be set to false.

When creating a plane without a texture but with different colors per texture createTextureCoordinates should be set to false and createVertexColorBuffer should be set to true.

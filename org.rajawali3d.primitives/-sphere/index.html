Sphere

open class Sphere : Object3D

A sphere primitive. The constructor takes two boolean arguments that indicate whether certain buffers should be created or not. Not creating these buffers can reduce memory footprint.

When creating solid color sphere both createTextureCoordinates and createVertexColorBuffer can be set to false.

When creating a textured sphere createTextureCoordinates should be set to true and createVertexColorBuffer should be set to false.

When creating a sphere without a texture but with different colors per texture createTextureCoordinates should be set to false and createVertexColorBuffer should be set to true.

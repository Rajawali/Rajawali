replaceCamera

open fun replaceCamera(camera: Camera!, location: Int): Boolean

Replaces a Camera in the renderer at the specified location in the list. This does not validate the index, so if it is not contained in the list already, an exception will be thrown. If the Camera being replaced is the one in current use, the replacement will be selected on the next frame.

Parameters

camera - Camera!: Camera object to add.

location - Int: Integer index of the camera to replace.

Return
Boolean: boolean True if the replacement was successfully queued.

open fun replaceCamera(oldCamera: Camera!, newCamera: Camera!): Boolean

Replaces the specified Camera in the renderer with the provided Camera. If the Camera being replaced is the one in current use, the replacement will be selected on the next frame.

Parameters

oldCamera - Camera!: Camera object to be replaced.

newCamera - Camera!: Camera object replacing the old.

Return
Boolean: boolean True if the replacement was successfully queued.

gluPerspective

open static fun gluPerspective(gl: GL10!, fovy: Double, aspect: Double, zNear: Double, zFar: Double): Unit

Set up a perspective projection matrix

Parameters

gl - GL10!: a GL10 interface

fovy - Double: specifies the field of view angle, in degrees, in the Y direction.

aspect - Double: specifies the aspect ration that determins the field of view in the x direction. The aspect ratio is the ratio of x (width) to y (height).

zNear - Double: specifies the distance from the viewer to the near clipping plane (always positive).

zFar - Double: specifies the distance from the viewer to the far clipping plane (always positive).

open fun addFromGeometry3D(offset: Vector3!, geometry: Geometry3D!, createVBOs: Boolean): Unit

Adds the geometry from the incoming geometry with the specified offset. Note that the offset is only applied to the vertex positions.

Parameters

offset - Vector3!: Vector3 containing the offset in each direction. Can be null.

geometry - Geometry3D!: Geometry3D to be added.

createVBOs - Boolean: boolean If true, create the VBOs immediately.
